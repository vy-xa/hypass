## Building instructions:

1 - Download/clone this repo

2 - Run `cd hypass`

3 - Run `cargo build --release`

4 - Thats it!

2 - In settings, go to custom integrations

3 - Click `New`

4 - Locate hypass.exe in your files

5 - Save!
